# Posts App - JSONPlaceholder API

Una aplicación React que consume la API de JSONPlaceholder y muestra los posts en pantalla.

## Características

- ✅ Consume la API de JSONPlaceholder (`https://jsonplaceholder.typicode.com/`)
- ✅ Muestra posts y información de usuarios
- ✅ Interfaz responsiva con diseño en grid
- ✅ Estados de carga y manejo de errores
- ✅ Diseño moderno con efectos hover
- ✅ Pruebas unitarias incluidas

## API Endpoints utilizados

- **Posts**: `GET https://jsonplaceholder.typicode.com/posts`
- **Usuarios**: `GET https://jsonplaceholder.typicode.com/users`

## Funcionalidades implementadas

1. **Obtención de datos**: Realiza llamadas simultáneas a las APIs de posts y usuarios
2. **Estado de carga**: Muestra un spinner mientras se cargan los datos
3. **Manejo de errores**: Muestra mensajes de error con opción de reintentar
4. **Visualización de datos**: Presenta los posts en tarjetas con información del autor
5. **Diseño responsivo**: Se adapta a diferentes tamaños de pantalla
